# CSS3 新属性

> 参考资料：[css3 新属性](https://github.com/febobo/web-interview/issues/106)

### 新样式

**css3 新增加了边框属性：**

- border-radius
- box-shadow
- border-image

**一些关于背景的属性：**

- background-clip
- background-origin
- background-size
- background-break

**颜色属性：**

新增了，rgba函数用来表示颜色，a 表示透明度

**transition属性**

**transform属性**

**animation属性**

### 新选择器

CSS3 中也新增加了一些选择器,例如下面这些

- :only-child
- :nth-child
- first-of-type
- a[src^='https']
